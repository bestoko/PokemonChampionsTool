# 项目需求说明

## 目标
本项目为“宝可梦冠军对战”工具型网站，优先实现核心对战辅助功能，后续可扩展更多对战内容。

## MVP 功能
1. 图鉴查询
   - 按名称、编号、属性查询宝可梦
   - 显示属性、种族值、特性、可选招式、形态/进化信息
2. 属性克制
   - 支持单属性和双属性输入
   - 显示弱点、耐性、免疫、倍率结果
3. 速度线
   - 多只宝可梦速度值对比
   - 显示速度排序、速度差、区间、加速/减速影响
4. 伤害计算器
   - 选择攻击宝可梦、目标宝可梦、招式
   - 支持高级规则：特性、道具、天气、场地、能力变化、护盾

## 用户场景
- 训练师希望快速查询某个宝可梦的信息。
- 训练师想判断两个属性组合的克制关系。
- 训练师希望比较几只宝可梦的速度差距。
- 训练师希望估算某招式对某目标的伤害范围。

## 约束和原则
- 使用静态数据优先，数据结构需可扩展。
- UI 风格保持简洁、淡蓝色调。
- 优先完成可用的 MVP，再进行迭代优化。

# 宝可梦冠军对战辅助工具

基于 Pokémon Champions Season M-2（Regulation M-A）的对战辅助网站，提供图鉴查询、属性克制、速度线、伤害计算、道具百科功能。

## 功能

- **图鉴查询** — 搜索 185 只当前赛季可用宝可梦，查看种族值、特性与招式（按物理/特殊/变化分类，悬停查看详情）
- **属性克制** — 选择 1-2 个防守属性，查看弱点 / 抵抗 / 免疫分类
- **速度线** — 查看所有宝可梦 Lv.50 速度面板：满速、极速、顺风后速度
- **伤害计算** — 标准对战公式，支持天气、场地、能力等级修正，招式按攻击者联动筛选
- **道具百科** — 110 个可用携带道具，按 S/A/B/C/Mega 等级分类浏览

## 技术

纯静态站点（HTML/CSS/JS），无框架依赖。数据从 `src/data/` 下的 JSON 文件异步加载。

## 本地运行

需通过 HTTP 服务器访问（直接打开 HTML 会因 fetch 跨域限制无法加载数据）：

```bash
npx serve src
# 或
python3 -m http.server -d src
```

## 数据来源

宝可梦数据基于 Pokémon Champions Season M-2（2026.05.13–06.17）官方阵容。图片来自 PokéAPI。
